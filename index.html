Interactive Neighbor Joining Algorithm

The Interactive Neighbor-Joining Algorithm seeks to provide an easy-to-use tool for visualizing the
Neighbor-Joining algorithm, given an additive matrix.

You can use the "Number of Nodes" input to determine the size of your matrix, and the "Nodes" input to determine the name of your nodes.

Generate Empty Matrix: Generate a new empty matrix based on size and sequece array.
Generate Phylogeny Tree: Generate the phylogeny tree based on the filled-in additive matrix.
Toggle Neighbor-Joining Order: Toggle text that gives overview of the order in which edges are created. Show Ordering Animation: Show animation that highlights the order in which nodes are clustered.
